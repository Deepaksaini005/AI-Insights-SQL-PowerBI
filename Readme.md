# 🤖 AI Prompt Analytics & BI Dashboard

A professional **SQL + Power BI** project focused on analyzing AI prompt usage, user engagement, token consumption, response quality, and AI model performance through interactive dashboards and business intelligence analytics.

---

## 📌 Project Overview

This project combines **SQL analytics** with **Power BI visualizations** to generate actionable insights from AI platform data.

The dashboard provides analysis on:

- User activity & engagement
- AI model performance
- Token consumption & API cost
- Prompt quality analysis
- Monthly usage trends
- Churn risk analysis
- Global user distribution

---

## 🛠️ Technologies Used

- MySQL
- SQL
- Power BI
- DAX
- Excel / CSV

## 📊 Dashboard Highlights

---

## 📈 Key Insights

- SQL and Research prompts achieved high quality scores
- GPT-4o and Gemini models showed highest usage
- Research category consumed maximum tokens
- Medium churn users formed the largest segment
- User engagement peaked during August and December
